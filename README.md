# 🔬 Sentinel — Early Cancer Detection Intelligence

> An AI-powered early cancer detection web app with visual skin analysis, symptom checker, and clinical AI consultation.

🌐 **Live Demo:** [sentinel-cancer.netlify.app](https://sentinel-cancer.netlify.app)

---

## ✨ Features

| Feature | Description |
|---|---|
| 📷 **Visual Skin Analysis** | Upload a photo of a mole or skin lesion — AI analyzes it using the clinical ABCDE melanoma framework |
| 💬 **AI Consultation** | Ask anything about cancer symptoms, risk factors, and screenings in plain language |
| ☑ **Symptom Checker** | Interactive 8-symptom checklist with a real-time risk indicator and age-based screening schedule |
| 🔑 **API Key Management** | Paste your Gemini key once — it's saved locally and remembered across sessions |

---

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Sarthak-67-cmd/sentinel-.git
   ```

2. **Open `Index.html`** directly in Chrome — no server or build step needed

3. **Get a free Gemini API key** at [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

4. **Paste the key** into the 🔑 field at the top of the app and press **✓**
